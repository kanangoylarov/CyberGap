# GRC Intelligence — Enterprise Assessment Platform

AI-powered GRC (Governance, Risk & Compliance) assessment platform built with Node.js, Express, SQLite, and Claude AI.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Configure environment
```bash
cp .env.example .env
```
Open `.env` and add your Anthropic API key:
```
ANTHROPIC_API_KEY=sk-ant-your-key-here
PORT=3000
```
Get your API key from: https://console.anthropic.com/

### 3. Start the server
```bash
npm start
```

### 4. Open in browser
```
http://localhost:3000
```

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/questions` | Get all 49 GRC questions |
| `POST` | `/api/assess` | Submit assessment → calls Claude API → saves to DB |
| `GET` | `/api/assessments` | List all assessments |
| `GET` | `/api/assessments/:id` | Get single assessment with full AI report |
| `DELETE` | `/api/assessments/:id` | Delete assessment |
| `GET` | `/api/stats` | Dashboard statistics |

## 📊 Database Schema

**assessments** table:
- `id` — Primary key
- `company_name` — Organization name
- `industry` — Sector/industry
- `contact_email` — Contact email
- `answers` — JSON: all 49 question answers
- `scores` — JSON: calculated domain and overall scores
- `overall_score` — Integer 0-100
- `risk_level` — "Low Risk" | "Medium Risk" | "High Risk"
- `maturity_level` — "Initial" | "Developing" | "Defined" | "Managed" | "Optimizing"
- `ai_report` — JSON: full Claude AI analysis
- `created_at` — Timestamp

## 🤖 AI Report Contents

Each report generated by Claude includes:
- Executive Summary (4-5 sentences)
- CEO Message (plain-language)
- Key Strengths with details
- Critical Findings with severity
- Domain Analysis (7 domains) with trend + quick wins
- 5+ Recommendations with:
  - Real-world industry examples
  - Step-by-step implementation (4-6 steps)
  - Specific tools & frameworks
  - Common pitfalls to avoid
  - Success metrics to track
  - Timeline, owner, cost, effort
- 4-Phase Implementation Roadmap (0-12 months)
- Risk Register
- Compliance Gap Analysis (GDPR, ISO 27001, etc.)
- Estimated Investment & ROI
- Industry Benchmark Comparison
